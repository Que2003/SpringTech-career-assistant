# Project Notes

## Portfolio angle

SpringTech Career Assistant is a practical front-end project that helps job seekers turn an IT job description into useful application material.

This project can be described on a resume like this:

- Built a browser-based career assistant that converts IT job descriptions into ATS keywords, resume bullet ideas, cover letter drafts, interview answers, and study plans.
- Implemented a static HTML/CSS/JavaScript application with browser localStorage for saving job opportunities.
- Designed a clean responsive user interface focused on help desk, desktop support, technical support, and cybersecurity job workflows.
- Created rule-based keyword extraction and structured content generation without requiring a paid API.

## Skills demonstrated

- Front-end development

- Career workflow automation
